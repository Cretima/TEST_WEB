과정

  1. 폴더(Direction)와 파일(.html)을 만든다
  2. >ctrl + o (window 기준), 파일 오픈으로 내 컴퓨터의 웹을 불러온다
  3. 아무글 끄적거리고 파일 저장, 띄어둔 웹에 F5(refresh) 하면 짜잔! 자신이 쓴 글이 나옵니다

 

 

기본지식

  1. 테그: 열린 TAG + 닫힌(/) TAG
  2. html은 접근성이 매우 중요, 비즈니스 측면, 휴머니즘
  3. title 이 본문의 제목은
  4. meta charset utf-8 : 이 본문이 utf-8로 정해져있다

 

html 기본 구조

  • !doctype html : 이 문서는 html로 이루어져있다
  • html 최상위 테그 head와 body를 감싸 사람과의 약속이야
  • head body의 상위 테그, 본문을 설명
  • body /body head의 하위 테그, 본문

 

문법

1. 문장 강조: <> 안에 strong + <> 안에 /strong
2. 밑줄이면: <> 안에 u(underline) + <> 안에 /u
3. 구획제목: h1 ~ h6
4. 줄바꿈 p /p and br
5. ul(unorderd list), ol(orderd list), li(list)
5.1 - ul, ol은 li를 같이 써야함. 또 html의 리스트(li)를 정의할때 사용
5.2 - ul, ol의 차이점은 앞에 넘버링 유무 차이
6. 테그의 심화 속성(attribute), img(이미지) 태그, src(source)
7. a(anchor) 흔히 말하는 링크이며 html의 황제
7-1. target ="_brank"는 링크를 클릭했을 때 새창에서 페이지 열림
7-2. title은 이 링크가 어떤 내용을 담고 있는지를 툴팁으로 보여주는 기능입니다.
7.3. > 링크 명 또는 title="..." > 어떤 글 에 링크 대입
